https://rails-onlinestore.herokuapp.com/

## Database creation
```rails db:migrate```
## Database initialization
```rails db:seed```

# Run(local)
```
cd store
bundle install --without production
rails server
```

## Deployment instructions
```
1. heroku create
2. git push heroku master
3. heroku run rails db:migrate
4. heroku run rails db:seed
```
